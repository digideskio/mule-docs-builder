CloudHub is an integration platform as a service (iPaaS). It allows you to deploy sophisticated cross-cloud integration applications, create new APIs on top of existing data sources, integrate on-premise applications with cloud services, and much more. With Mule Studio – the user-friendly development tool – you can quickly build these integration applications and deploy to CloudHub with just a few clicks. This guide will help you learn the basics and get started developing your first applications.

Learn CloudHub Basics

  • Download Anypoint Studio.

  • Read our CloudHub at a Glance guide to learn the basic concepts.

  • Follow our easy Hello World on CloudHub tutorial to get your first integration application up and running with Anypoint Studio and CloudHub.

  • Then, get started with connectors to connect to external services like Twitter.

Still don’t understand how CloudHub can be of use to you? Read more about things you can do with CloudHub.
