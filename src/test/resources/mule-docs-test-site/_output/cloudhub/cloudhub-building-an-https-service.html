Building an HTTPS service

CloudHub supports building HTTPS/SSL based services so you can be ensured of your data confidentiality. It is very easy to configure your service to support SSL.

This tutorial builds on existing tutorials, so to begin, you’ll need to build a service using the Hello World tutorial.

You can now modify your existing service to use HTTPS as detailed in the following steps.

First, if you don’t have a keystore for your service, you’ll need to generate a new keystore. You can do this using the command line and the keytool utility from the JDK.

keytool -genkey -alias mule -keyalg RSA -keystore keystore.jks

This command will create a file named keystore.jks. Before proceeding, verify that this file exists and that it is located in the folder src/main/resources.
